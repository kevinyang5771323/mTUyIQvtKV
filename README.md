# Python数字图书馆系统 python136

## 项目概述

本项目是基于Python语言的数字图书馆系统，采用Django框架进行开发，前端使用HTML、CSS和JavaScript技术，后端数据库使用MySQL。系统实现了用户登录注册、图书搜索、借阅和归还等核心功能，并包含借书排行榜和最受欢迎图书的统计。

## 技术栈

- 后端框架：Django
- 数据库：MySQL
- 前端技术：HTML、CSS、JavaScript

## 功能模块

- 用户模块
  - 登录
  - 注册
- 图书检索模块
  - 图书搜索
  - 图书详情
- 借阅管理模块
  - 图书借阅
  - 图书归还
- 统计模块
  - 借书排行榜
  - 最受欢迎图书

## 系统角色

- 系统管理员：负责图书信息维护和系统管理
- 普通用户：进行图书搜索、借阅和归还

## 运行环境

- Python环境
- Django框架
- MySQL数据库
- 支持HTML、CSS、JavaScript的浏览器

## 部署步骤

1. 配置Python环境
2. 安装Django框架
3. 配置MySQL数据库
4. 运行项目
